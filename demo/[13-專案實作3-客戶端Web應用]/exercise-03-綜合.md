# 練習 03 — 綜合挑戰：完整 Dashboard + 部署

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
完成 Freelancer Dashboard 的核心功能並部署到 Vercel + Neon。

## 要求

1. **完成一個完整的 CRUD 功能**
   選擇「客戶管理」或「專案管理」，完成從前端到資料庫的完整功能：
   - 列表頁（含搜尋 + 分頁）
   - 新增表單（含即時驗證）
   - 編輯功能（預填現有資料）
   - 刪除功能（確認對話框）

2. **實作認證**
   使用 NextAuth v5 Credentials provider：
   - 登入頁面
   - 登出功能
   - 未登入自動重導向到登入頁
   - 已登入自動重導向到 Dashboard 首頁

3. **加入統計圖表**
   在 Dashboard 首頁加入：
   - 本月收入（數字 + 環比增減）
   - 進行中專案數量
   - 本週工時（可用簡單的 bar chart 或只顯示數字）

4. **部署到 Vercel + Neon**
   - 設定 Neon 資料庫（免費方案）
   - 在 Vercel 設定所有必要的環境變數
   - 執行資料庫遷移（Drizzle migrate）
   - 部署並驗證功能正常

## 端對端驗收測試
按以下流程手動測試，確認每步都成功：
1. 造訪首頁 → 自動重導向到登入頁
2. 用測試帳號登入 → 成功進入 Dashboard
3. 新增一個客戶
4. 為該客戶新增一個專案
5. 查看統計數字有沒有更新
6. 登出 → 自動重導向到登入頁

## 完成標準
- [ ] 一個完整 CRUD 功能可以使用
- [ ] 認證流程正常
- [ ] Vercel 連結可以訪問
- [ ] 端對端測試 6 步全部通過
