# 練習 03 — 綜合挑戰：設計你的第一個工作流

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
設計一個「每天早上自動整理 workspace/ 下所有 .log 檔案，移到 logs/YYYY-MM-DD/ 資料夾」的 agentic 工作流。

不需要完整實作，需要完成以下設計文件：

1. **工具清單**：這個工作流需要哪些工具（Read / Write / Bash / 其他）？每個工具在哪一步被使用？

2. **步驟分解**：描述每一步 Claude Code 應該做什麼（用「首先…然後…最後…」的格式）

3. **失敗點分析**：識別至少 3 個可能的失敗點，每個失敗點說明：
   - 什麼情況下會失敗？
   - 如何偵測到失敗？
   - 失敗後應該怎麼處理？

4. **擴展思考**：如果要擴展成「整理 workspace/ 下所有類型的暫存檔（.log / .tmp / .bak）」，設計需要改哪些地方？

## 完成標準
- [ ] 工具清單完整，說明每個工具的使用時機
- [ ] 步驟分解清楚，可以讓另一個人照著執行
- [ ] 找到至少 3 個失敗點並有處理方案
- [ ] 解答通過自我審查

## 完成後
將解答存入 `answer/ex03-answer.md`
