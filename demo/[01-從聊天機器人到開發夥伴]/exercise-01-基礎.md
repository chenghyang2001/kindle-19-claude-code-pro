# 練習 01 — 基礎：比較兩種工作模式

## 情境說明
你剛安裝好 Claude Code CLI，想了解它和平常用 Claude Chat 有什麼不同。

## 任務

### 任務 1：兩種模式實測
用 Claude Chat 輸入「請幫我建立一個 hello.py」，記錄它的回應方式。
再用 Claude Code 做同樣的事，記錄實際發生什麼（檔案有沒有真的被建立？）

比較：
- Claude Chat 給了什麼？
- Claude Code 做了什麼？
- 差異在哪裡？

### 任務 2：識別工具呼叫
觀察 Claude Code 完成任務時，它呼叫了哪些工具？
（提示：注意 Read / Write / Bash 等工具名稱出現的位置）

列出：
1. 工具名稱
2. 每個工具被呼叫時做了什麼
3. 如果少了某個工具，任務會失敗嗎？

## 完成後
將解答存入 `answer/ex01-answer.md`
