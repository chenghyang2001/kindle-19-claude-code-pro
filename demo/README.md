1. 從 `[01-從聊天機器人到開發夥伴]` 開始

3. 執行 `/study-scaffold carry N` 攜帶答案到下一課
4. 填寫 `STEP_LOG.md` 記錄學習過程
5. 完成所有課程後，`docs/踩坑紀錄.md` 會是最寶貴的筆記
