# 練習 03 — 綜合挑戰：Prompt 效果評測

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
對同一個任務，設計 3 個不同的 Prompt，評測哪個效果最好。

## 評測任務
選擇以下任一任務作為評測目標：
- 「為一個 React 元件寫 TypeScript 測試」
- 「找出這段 Python 程式碼的安全問題」
- 「解釋這段 SQL Query 在做什麼」

## 評測流程

### 第一步：設計 3 個 Prompt 變體
- **Prompt A**：簡單直白（只說要做什麼）
- **Prompt B**：加角色 + 格式要求
- **Prompt C**：加角色 + 格式 + 範例 + 負面約束

### 第二步：執行並記錄
對同樣的程式碼，分別用 3 個 Prompt，記錄：
- Claude 的回答長度（行數）
- 回答的格式是否符合預期
- 回答的品質（你的主觀評分 1-5）
- 有沒有錯誤或誤解

### 第三步：分析差異
- 哪個 Prompt 最有效？為什麼？
- 加了角色/格式/範例後，效果有提升嗎？
- 複雜 Prompt 是否一定比簡單的好？

## 完成標準
- [ ] 3 個 Prompt 都執行過
- [ ] 有量化的比較表格
- [ ] 有結論：什麼情況用什麼類型的 Prompt

## 完成後
將評測結果 + 結論存入 `answer/ex03-answer.md`
