# 練習 01 — 基礎：撰寫 CLAUDE.md

## 情境說明
你要為一個新的 Python Flask API 專案建立 CLAUDE.md，讓 Claude Code 一進入就知道專案規則。

## 任務

### 任務 1：最小可用 CLAUDE.md
為一個 Flask API 專案撰寫最小可用的 CLAUDE.md，必須包含以下六個區塊：

1. **專案說明**：這是什麼、解決什麼問題
2. **技術堆疊**：語言版本 + 框架 + 資料庫 + 部署平台
3. **執行方式**：如何啟動開發伺服器、如何跑測試
4. **目錄結構**：主要資料夾的用途
5. **禁止事項**：絕對不能做的操作（例如：不能直接修改 prod DB）
6. **常用指令**：每天都會用到的 3-5 個 CLI 指令

### 任務 2：測試有效性
建立 CLAUDE.md 後，在新的 Claude Code session 中問：
- 「這個專案是做什麼的？」
- 「我要怎麼跑測試？」
- 「這個專案有什麼操作是被禁止的？」

確認 Claude Code 能從 CLAUDE.md 中正確引用資訊，並記錄哪些問題答不好。

## 完成後
將 CLAUDE.md 內容 + 測試結果存入 `answer/ex01-answer.md`
