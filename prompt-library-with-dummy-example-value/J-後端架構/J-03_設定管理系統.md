# J-03　設定管理系統

> **類別**：J. 後端架構

---

```
為 部落格平台（Next.js + Neon PostgreSQL） 設計設定管理系統：
環境：development / staging / production
設定項目：DATABASE_URL（敏感）、NEXTAUTH_SECRET（敏感）、NEXT_PUBLIC_APP_URL（公開）、STRIPE_SECRET_KEY（敏感）
要求：
- 啟動時驗證所有必要設定（缺少就 crash）
- 敏感值只從環境變數讀取（不寫死）
- 提供 .env.example 範本
- 型別安全（TypeScript 或 Pydantic）
```
