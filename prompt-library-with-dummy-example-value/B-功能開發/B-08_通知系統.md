# B-08　通知系統

> **類別**：B. 功能開發

---

```
實作 Email + 站內通知 通知系統：
觸發事件：new_comment（新留言）、article_published（追蹤作者發文）、welcome（註冊歡迎）
通知範本：• new_comment: {{commenter}} 在《{{article_title}}》留言了
• article_published: 你追蹤的 {{author}} 發布了《{{title}}》
• welcome: 歡迎加入！帳號 {{email}} 已建立成功
要求：
- 非同步發送（不阻塞主要流程）
- 失敗重試（最多 3 次，指數退避）
- 記錄發送日誌（成功/失敗）
```
