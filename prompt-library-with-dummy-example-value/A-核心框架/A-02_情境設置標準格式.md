# A-02　情境設置標準格式

> **類別**：A. 核心框架

---

```
在開始之前，請先理解以下情境：
- 專案類型：Web App（全端）
- 技術堆疊：TypeScript, Next.js 16 App Router, Neon PostgreSQL
- 目前問題：文章列表頁在資料超過 1,000 筆時首次載入需 5 秒以上，使用者回報頁面嚴重卡頓
- 目標：首次載入降至 1 秒以內，捲動流暢（60fps），支援每頁 20 筆 cursor-based 分頁
- 限制條件：不能修改 DB Schema（沒有遷移視窗）；需相容 Next.js 14+；不引入新的大型依賴
請先複述你的理解，確認後再開始動手。
```
