# 解答格式規則（按需載入）

> 只在存取 answer/ 目錄內的檔案時載入。

## 解答檔案必要結構

每個 `exNN-answer.md` 必須包含：

```markdown
# Exercise NN 解答 — [練習標題]

## 任務 1：[任務名稱]
[解答內容]

---

## 任務 2：[任務名稱]
[解答內容]

---

## 學習洞察
[這個練習最重要的一個概念，用自己的話說明]
```

## 品質標準

- **任務完整性**：每個 exercise 的任務都必須有對應解答，不可跳過
- **可引用性**：解答應能被 Claude 在新 session 中直接引用，回答相關問題
- **真實性**：解答必須反映實際理解，不是抄書。「學習洞察」用自己的話

## 程式碼解答格式

含程式碼的解答用 fenced code block + 語言標示：

```python
# Python 範例
```

```json
// JSON 範例
```

## 禁止事項

- 禁止把 exercise 題目原文複製到解答（浪費 context）
- 禁止只寫「參考書本 XXX 頁」（無效解答）
- 禁止解答超過 200 行（分拆到多個概念說明）
