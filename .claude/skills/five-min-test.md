# /five-min-test — CLAUDE.md 五分鐘健康檢查

快速驗證當前專案的 CLAUDE.md 是否真正對 Claude Code 有效。

## 步驟

1. 讀取當前專案的 CLAUDE.md
2. 依照「六大支柱」逐項評分（0-2 分）：
   - [ ] **技術堆疊**：框架+版本是否寫清楚？
   - [ ] **執行方式**：Claude 能否照著啟動？
   - [ ] **架構決策**：「為什麼選 A 不選 B」有沒有？
   - [ ] **限制條件**：「不能做什麼」有沒有寫？
   - [ ] **模組引用**：有用 `@instructions/` 引用全域規則嗎？
   - [ ] **目錄結構**：和實際檔案結構一致嗎？

3. 輸出評分報告：
   - ✅ PASS（≥10 分）：CLAUDE.md 品質良好
   - ⚠️ 需改善（6-9 分）：列出待改項目
   - ❌ 重寫（< 6 分）：建議重新撰寫

4. 若有 ⚠️ 或 ❌，列出具體改進建議（不超過 5 條）
