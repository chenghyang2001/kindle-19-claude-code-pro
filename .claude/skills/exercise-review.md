# 提示詞範本：練習解答審查

> 可重用提示詞。完成練習後使用，請 Claude 給出評分與改進建議。

---

## 使用方式

完成 `answer/exNN-answer.md` 後，把以下提示詞貼給 Claude：

---

**提示詞（複製貼上）：**

```
請審查我剛完成的 [課程NN] [練習NN] 解答（answer/exNN-answer.md）。

評審標準：
1. 內容完整性 — 是否涵蓋練習要求的所有任務？
2. 概念理解 — 是否正確掌握本課的核心概念？
3. 設計品質 — 如果是程式碼/架構，有無明顯缺失？
4. 實用性 — 這個解答在真實專案中是否可直接採用？

請給出：
- 整體評分（⭐ 1-5）
- 每個標準的具體回饋
- 最多 2 個改進建議（不要超過，避免改動範圍過大）
```

---

## 快速版本（單一問題）

```
這個 CLAUDE.md 的禁止事項設計夠好嗎？從「防止 Claude 做危險操作」的角度評估。
```

```
這個 Hook 的靜默失敗設計是否完整？會有哪些情境下失敗但沒有任何提示？
```

```
如果這個 MCP 配置出現 token 過期，Claude 會怎麼處理？現在的設計有沒有對應？
```
